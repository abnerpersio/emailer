<h1 align="center">✉️ Emailer</h1>

Este é um serviço de emails simples que usa um sistema de filas para fazer disparos de emails

### Features existentes:

- Docker
- Enviar um email
- Envio de email em lote

### Futuras features:

- Criação de templates HTML para envio

### Tecnologias usadas:

- Node.JS
- Typescript
- Express
- Nodemailer
- Redis
- Bull
